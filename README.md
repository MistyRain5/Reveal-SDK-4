# Installation
```ruby
target 'YourMainAppTargetName' do
     pod 'Reveal-SDK', :git=>"https://github.com/MistyRain5/Reveal-SDK-4.git", :configurations => ['Debug']
end
```
Then, run the following command:

```bash
$ pod install
```
